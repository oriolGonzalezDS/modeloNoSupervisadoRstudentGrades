# modeloNoSupervisadoRstudentGrades

Proyecto escolar para en el que con R trabajamos el prepoceado, escaldo y aplicación de las componetntes principales para llevar a cabo distintos
modelos predictivos basados en modelos de aprendizaje no supervisados. Queremos descubrir cuales son las principales causas de fracaso escolar en las escuelas
portuguesas, para esto se estudia mediante reglas de asociación las notas de sus alumnos en dos asignaturas.
